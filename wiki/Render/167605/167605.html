The DateRangeValidation attribute is applied to a Date-type property of the plug-in class. It allows the plug-in creator to specify that the value of the property must fall within a certain range. If the configured value is not within that range, appropriate error messaging is displayed in the UI.

Example:

public class DisclaimerSelectorPlugin
{
    [DateRangeValidation(new DateTime(1999,1,15), DateTime.Now)]
    ...
    public DateTime DisclaimerDate
    {
        get;
        set;
    }
...
}
