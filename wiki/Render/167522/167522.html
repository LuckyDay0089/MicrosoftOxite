The AuthorUrls attribute is applied to the plug-in class. It allows the plug-in creator to specify homepage url(s) for each of the people listed in the Authors attribute. There should be one entry in the AuthorUrls attribute for every Authors entry.

Example:

[Authors("Travis Merkel", "Steve Perry")]
[AuthorUrls("http://lostincompilation.com", "http://www.journeymusic.com")]
...
public class DisclaimerSelectorPlugin
{
...
}
