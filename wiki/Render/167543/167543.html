The IconXmallDisabled attribute is applied to the plug-in class. It allows the plug-in creator to specify a small icon to displayed in association with the plug-in in the UI. This icon will be displayed when the plug-in is in a disabled state. This can either be an absolute path or the simple file name of the icon if the icon file is located in the images directory of the plug-in folder structure.



Example:

[IconSmallDisabled("IconSmallDisabled.png")]
...
public class DisclaimerSelectorPlugin
{
...
}
